# Programa Lanzadera Sureste – GitHub Pages

Sitio estático (HTML/CSS/JS) para presentar el proyecto **Programa Lanzadera Sureste**.

## Estructura
- `index.html`
- `styles.css`
- `script.js`
- `assets/fonts/` tipografía local (descargas)
- `assets/images/` imágenes del proyecto
- `assets/Memoria_y_Proyecto_Programa_Lanzadera_Sureste_SUIZO_INSTITUCIONAL.docx` descarga

## Despliegue en GitHub Pages
1. Crear repo en GitHub (p.ej. `programa-lanzadera-sureste`).
2. Subir el contenido de `programa-lanzadera-sureste-site/` a la raíz del repo.
3. Settings → Pages → Deploy from a branch → `main` / root.
